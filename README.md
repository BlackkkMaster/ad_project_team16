# Анализ Данных  // Команда 16 // Самокаты

## Анализ спроса сервиса кикшеринга

### 1. Клонируйте репозиторий

```bash
git clone https://github.com/BlackkkMaster/ad_project_team16
cd ad_project_team16
```

### 2. Загрузите датасеты в папку [datasets](datasets)

### 3. Создайте и активируйте виртуальное окружение

- На Linux/macOS:

    ```bash
    python3 -m venv .venv
    source .venv/bin/activate
    ```

- На Windows:

    ```bash
    python -m venv .venv
    .venv\Scripts\activate
    ```

### 4. Установите зависимости

```bash
pip install -r ./requirements.txt
```

### 5. Откройте тетрадку - [scooter_sharing.ipynb](./scooter_sharing.ipynb)

- Jupyterlab:

```bash
jupyter lab
```

- Jupyter Notebook:

```bash
jupyter notebook
```
